# sanchez-post1-u3

Laboratorio: Página de Perfil con CSS3

# Descripción

Este proyecto es una página de perfil personal desarrollada para la asignatura de Programación Web. La web integra una estructura HTML5 semántica con un diseño estilizado en CSS3, destacando el uso de posicionamiento fijo para la navegación, una tarjeta de presentación con un distintivo posicionado absolutamente y un formulario de contacto accesible. El objetivo principal es aplicar de forma práctica conceptos de Box Model, selectores avanzados y estados interactivos para crear una interfaz funcional y profesional.
